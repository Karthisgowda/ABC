# Git Setup

For commits to appear on the correct GitHub profile, the repository author
email should match a verified GitHub email or GitHub noreply address.

Current recommended local configuration for this repo:

```bash
git config user.name "Karthisgowda"
git config user.email "192409624+Karthisgowda@users.noreply.github.com"
```

Check the active identity:

```bash
git config user.name
git config user.email
```
